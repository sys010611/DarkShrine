## プロジェクト概要

日本旅行での経験をもとに制作した、初めてのゲーム作品です。
初作品でありながら Steam で約 200 本を販売し、国内の著名な YouTuber にもプレイしていただくなど、大きな励みとなる成果を得ることができました。
